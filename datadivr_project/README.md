# DataDiVR Complete Project Export

Copy `Pangenome_Housekeeping_Stacked_Trees/` into:

```text
DataDiVR_WebApp/static/projects/
```

Then start the DataDiVR backend and open:

```text
http://127.0.0.1:3000/preview
```

Select `Pangenome_Housekeeping_Stacked_Trees`.

This export is paired with the full analysis objects under `outputs_3d/`,
including:

- `Pangenome_Housekeeping_Stacked_Trees_networkx_scenes.pkl`
- `Pangenome_Housekeeping_Stacked_Trees.json`
- `datadivr_coordinate_mappings.json`
- `networks_3d/`
- `networks_population_expanded/`
- layout and DataDiVR audit JSON files
